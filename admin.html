🚀 HOW TO UPLOAD IMAGES:

  1. Go to your GitHub repository
  2. Navigate to assets/projects/ (or gallery/team/)
  3. Click "Add file" → "Upload files"
  4. Drag & drop your images
  5. Commit directly to main branch

📋 INSTRUCTIONS:

1. Add images to your GitHub repo in the correct folders (/assets/projects/, /assets/gallery/, /assets/team/)
2. Use this admin panel to add entries with the correct image paths
3. Export data.json and commit it to your repo root
